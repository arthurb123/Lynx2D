Lynx2D GameObject

new GameObject(sprite: Sprite, x: number, y: number, w: number, h: number)
Parameters
sprite (Sprite) The sprite of the GameObject, can be undefined.
x (number) The X position.
y (number) The Y position.
w (number) The width.
h (number) The height.

Sprite

lx.Sprite - Lynx2D Sprite

new Sprite(source: string, c_x: number, c_y: number, c_w: number, c_h: number, cb: function)
Parameters
source (string) The source to an image file (can also be a HTML canvas).
c_x (number) The clip x can be a number, a callback function for once the sprite has loaded, or be left undefined.
c_y (number) The clip y position, can be left undefined.
c_w (number) The clip width, can be left undefined.
c_h (number) The clip height, can be left undefined.
cb (function) a callback function for once the sprite has loaded, can be left undefined.
